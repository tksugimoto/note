クリップボードの中身を便利に処理するツール(Progressive web apps, PWA)

できること

クリップボードに入っているテキストで
  • Twitterで検索
  • Google検索

使い方

  1. (事前準備) このPWAをインストール (Android & Chrome で動作確認済み)
  2. クリップボードに何か（テキスト）をコピーする
  3. このアプリを開く
  4. 「クリップボードを読み取る」をクリック
